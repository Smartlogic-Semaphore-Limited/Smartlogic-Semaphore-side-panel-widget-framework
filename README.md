# WorkbenchWidgetApi 
Library helper to integrate widgets with Smartlogic Semaphore Workbench

## Usage: 
1 - refer to library:

```html
<script src="dist/swApiWidget.min.js"></script>
```

2 - create new instance of widget

```javascript
var widget = new WorkbenchWidgetApi();
```

This will send to Host application the confirmation that widget is ready.
